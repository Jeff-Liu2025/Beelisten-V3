# 长音频训练优化与新训练模式 Spec

## Why
CET4等考试真题音频长达20多分钟，现有的词块填空训练模式存在以下问题：
1. 训练时间过长，用户难以一次性完成
2. 训练进度无法保存，中断后需要重新开始
3. 缺乏针对考试真题的专项训练模式

## What Changes
- 新增音频分段功能，将长音频拆分为多个训练单元
- 新增训练进度保存功能，支持断点续训
- 新增"听力选择"训练模式，模拟考试题型
- 新增"听写训练"模式，逐句听写
- 新增"跟读训练"模式，录音对比评分
- 优化词块填空训练，支持分段和进度保存

## Impact
- Affected specs: 训练模块、存储模块、播放器模块
- Affected code: 
  - `js/training/word-fill.js`
  - `js/word-fill-training.js`
  - `js/word-fill-select.js`
  - `js/store/index.js`
  - `css/training.css`

## ADDED Requirements

### Requirement: 音频分段功能
系统应支持将长音频自动或手动分段，便于分段训练。

#### Scenario: 基于内容的智能分段（推荐）
- **WHEN** 用户选择一个考试类音频资源（如CET4）
- **THEN** 系统自动识别SRT中的分段标记（Section A/B/C、News report、Conversation、Passage等）
- **AND** 按照实际考试结构分段
- **AND** 显示分段名称（如"Section A - News Report 1"）

#### Scenario: 自动分段（普通音频）
- **WHEN** 用户选择一个时长超过5分钟的普通音频资源
- **THEN** 系统自动按字幕段落将音频分成多个训练单元
- **AND** 每个单元包含3-5个句子
- **AND** 显示分段列表供用户选择

#### Scenario: 手动分段
- **WHEN** 用户点击"自定义分段"按钮
- **THEN** 用户可以手动选择训练的起始和结束时间

### Requirement: 训练进度保存
系统应支持训练进度的本地保存和恢复。

#### Scenario: 保存进度
- **WHEN** 用户完成一个训练单元或主动点击"保存进度"
- **THEN** 系统将当前训练进度保存到localStorage
- **AND** 记录资源ID、当前段落索引、正确率等信息

#### Scenario: 恢复进度
- **WHEN** 用户重新进入训练页面
- **THEN** 系统检测是否有未完成的训练
- **AND** 提示用户是否继续上次训练

### Requirement: 听力选择训练模式
系统应提供听力选择题训练模式，模拟考试题型。

#### Scenario: 开始训练
- **WHEN** 用户选择"听力选择"模式
- **THEN** 系统播放一段音频后暂停
- **AND** 显示与该段内容相关的选择题
- **AND** 用户选择答案后显示正确答案和解析

#### Scenario: 题目生成
- **WHEN** 系统需要生成选择题
- **THEN** 根据字幕内容自动生成问题
- **AND** 问题类型包括：细节理解、主旨大意、推理判断

### Requirement: 听写训练模式
系统应提供逐句听写训练模式。

#### Scenario: 听写流程
- **WHEN** 用户选择"听写训练"模式
- **THEN** 系统播放一个句子后暂停
- **AND** 用户在输入框中输入听到的内容
- **AND** 系统对比用户输入与原文，高亮差异

#### Scenario: 提示功能
- **WHEN** 用户点击"提示"按钮
- **THEN** 显示首字母或部分字母提示

### Requirement: 跟读训练模式
系统应提供跟读训练模式，支持录音对比。

#### Scenario: 跟读流程
- **WHEN** 用户选择"跟读训练"模式
- **THEN** 系统播放一个句子
- **AND** 用户点击"开始录音"跟读
- **AND** 系统录制用户发音并对比评分

### Requirement: 训练统计与成就
系统应提供训练统计和成就系统。

#### Scenario: 统计数据
- **WHEN** 用户完成训练
- **THEN** 更新学习统计数据（训练次数、正确率、学习时长）
- **AND** 在"个人档案"页面显示统计图表

## MODIFIED Requirements

### Requirement: 词块填空训练优化
现有词块填空训练需要支持分段和进度保存。

#### Scenario: 分段训练
- **WHEN** 用户选择长音频进行词块填空训练
- **THEN** 系统自动将训练分为多个单元
- **AND** 用户可以单独完成每个单元
- **AND** 完成一个单元后自动保存进度

#### Scenario: 进度恢复
- **WHEN** 用户重新进入未完成的词块填空训练
- **THEN** 系统提示"检测到未完成的训练，是否继续？"
- **AND** 用户选择继续后从上次中断处恢复
