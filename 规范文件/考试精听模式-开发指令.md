# 考试精听模式交互流程优化 — 开发指令

## 背景

`js/quiz-training.js` 是考试听力训练页面。当前的问题是：音频播放到每道题的 `endTime` 就暂停并出题，但真实考试流程是**先连续播完整个 report（含问题音频），再统一显示这个 report 的所有题目**。

## 现有数据结构

### 1. Segment（音频分段）
由 `js/utils/audio-segment.js` 的 `smartSegment()` 从 SRT 字幕生成。对于 CET-4 资源，会按 `News Report One/Two/Three`、`Conversation One/Two`、`Passage One/Two/Three` 自动分段。

每个 segment 示例：
```js
{
  id: 'segment-0',
  type: 'newsReport',
  name: 'News Report 1',
  startTime: 43.62,   // report正文开始
  endTime: 160.6,     // 下一篇report标记之前
  duration: 116.98,
  subtitles: [...]    // 该report的所有字幕
}
```

### 2. 考试题目（questions.json）
```js
{
  "sections": [{
    "sectionId": "section-a",
    "sectionType": "news-report",
    "questions": [
      { "questionNumber": 1, "startTime": 123.82, "endTime": 126.54, "questionText": "...", "options": [...], "correctAnswer": "A", "explanation": null },
      { "questionNumber": 2, "startTime": 142.32, "endTime": 146.68, "questionText": "...", "options": [...], "correctAnswer": "B", "explanation": null }
    ]
  }]
}
```

### 3. 题目与 Segment 的对应关系
`startSegment()` 里已有过滤逻辑：
```js
const segmentQuestions = this.examQuestions.filter(q => 
    q.startTime >= this.currentSegment.startTime && 
    q.startTime < this.currentSegment.endTime
);
```
这个逻辑已经能正确把题目归到对应的 report 下（如 News Report 1 对应 Q1-Q2）。**不需要改数据结构**。

---

## 目标交互流程（考试模式）

```
播放 News Report 1 全部音频（正文 + 问题引导 + Question 1 + Question 2）
    ↓
播完整个 segment 后暂停
    ↓
统一显示 News Report 1 的所有题目（Q1、Q2）
    ↓
用户逐题作答（先答Q1，答完显示解析，再答Q2，答完显示解析）
    ↓
全部答完后，显示"下一篇"按钮
    ↓
点击后播放 News Report 2 全部音频
    ↓
...
```

### 修改点 1：onTimeUpdate — 移除"逐题暂停"逻辑

**当前问题代码（约第232-242行）：**
```js
if (this.currentSegmentQuestions && this.currentSegmentQuestions.length > 0) {
    const nextQuestion = this.currentSegmentQuestions[this.currentQuestionIndex];
    if (nextQuestion && currentTime >= nextQuestion.endTime) {
        this.audio.pause();
        this.isPlaying = false;
        this.questionPauseTime = currentTime;
        this.updatePlayButton();
        this.showQuestion(nextQuestion);
    }
}
```

**改为：**
```js
// 考试资源：不逐题暂停，等整个 segment 播完再统一出题
// 非考试资源：保持原有逻辑（如果需要）
if (!this.isExamResource && this.currentSegmentQuestions && this.currentSegmentQuestions.length > 0) {
    const nextQuestion = this.currentSegmentQuestions[this.currentQuestionIndex];
    if (nextQuestion && currentTime >= nextQuestion.endTime) {
        this.audio.pause();

```

**改为：**
- 如果是考试资源且有题目，调用新的 `showSegmentQuestions()` 方法统一显示当前 segment 的所有题目

```js
        // ...
    } else {
        this.showFinalResult();
    }
}
```

### 修改点 3：新增 showSegmentQuestions() 方法

这个方法负责**统一显示当前 segment 的所有题目**，并管理逐题作答流程。

```js
showSegmentQuestions() {
    this.hasAnswered = false;
    this.currentQuestionIndex = 0;
    this.correctCount = 0;
    this.wrongCount = 0;
    this.wrongAnswers = [];
    
    const optionsContainer = document.getElementById('quizOptionsContainer');
    if (!optionsContainer) return;
    
    // 渲染所有题目列表（可以折叠或只显示当前题）
    this.renderQuestionList();
    optionsContainer.classList.remove('hidden');
    
    // 显示第一题
    this.showQuestionAtIndex(0);
}
```

### 修改点 4：新增 renderQuestionList() 方法

在 UI 上渲染当前 segment 的所有题目（供用户查看进度）：

```js
renderQuestionList() {
    const container = document.getElementById('quizOptionsContainer');
    const total = this.currentSegmentQuestions.length;
    
    // 顶部显示进度：News Report 1 - 共2题
    const headerHtml = `
        <div class="segment-question-header">
            <h3>${this.currentSegment.name} — 共 ${total} 题</h3>
            <div class="question-progress-dots">
                ${this.currentSegmentQuestions.map((q, i) => `
                    <span class="progress-dot ${i === 0 ? 'current' : ''}" data-index="${i}">${i + 1}</span>
                `).join('')}
            </div>
        </div>
    `;
    
    // 题目区域
    const questionsHtml = `
        <div id="segmentQuestionsArea">
            <!-- 当前题目会在这里动态渲染 -->
        </div>
    `;
    
    container.innerHTML = headerHtml + questionsHtml;
}
```

### 修改点 5：新增 showQuestionAtIndex(index) 方法

显示指定索引的题目，替代原有的 `showQuestion()`（或复用 `showQuestion()` 但修改其渲染目标）。

```js
showQuestionAtIndex(index) {
    if (index < 0 || index >= this.currentSegmentQuestions.length) return;
    
    this.currentQuestionIndex = index;
    this.hasAnswered = false;
    this.currentQuestion = this.currentSegmentQuestions[index];
    
    // 更新进度点样式
    document.querySelectorAll('.progress-dot').forEach((dot, i) => {
        dot.classList.toggle('current', i === index);
    });
    
    // 更新进度文字
    const progressText = `${this.currentSegment.name} | 第 ${index + 1}/${this.currentSegmentQuestions.length} 题`;
    document.getElementById('trainingProgress').textContent = progressText;
    
    // 渲染当前题目
    const area = document.getElementById('segmentQuestionsArea');
    if (!area) return;
    
    const question = this.currentQuestion;
    
    area.innerHTML = `
        <div class="exam-question-header">
            <div class="exam-question-number">Question ${question.questionNumber}</div>
            <div class="exam-question-text">${question.questionText}</div>
        </div>
        <div class="exam-options-list">
            ${question.options.map((option, i) => `
                <div class="quiz-option" data-index="${i}" data-correct="${option.correct}">
                    <span class="option-label">${option.label}</span>
                    <span class="option-text">${option.text}</span>
                </div>
            `).join('')}
        </div>
        <div id="questionResultArea"></div>
        <div id="nextQuestionControl" style="display:none; margin-top: 16px;">
            <button class="training-action-btn" id="nextQuestionBtn">
                ${index < this.currentSegmentQuestions.length - 1 ? '下一题 →' : '完成，进入下一篇 →'}
            </button>
        </div>
    `;
    
    // 绑定选项点击
    area.querySelectorAll('.quiz-option').forEach(optionEl => {
        optionEl.addEventListener('click', () => {
            if (!this.hasAnswered) {
                this.selectOption(optionEl);
            }
        });
    });
    
    // 绑定下一题按钮
    const nextBtn = document.getElementById('nextQuestionBtn');
    if (nextBtn) {
        nextBtn.addEventListener('click', () => this.goToNextQuestion());
    }
    
    this.currentOptions = question.options;
}
```

### 修改点 7：新增 goToNextQuestion() 方法

```js
goToNextQuestion() {
    const nextIndex = this.currentQuestionIndex + 1;
    
    if (nextIndex < this.currentSegmentQuestions.length) {
        // 还有下一题，显示下一题
        this.showQuestionAtIndex(nextIndex);
    } else {
        // 当前 segment 所有题目答完，进入下一 segment
        this.hideQuestionUI();
        
        if (this.currentSegmentIndex < this.segments.length - 1) {
            // 显示过渡提示，让用户点击继续
            const optionsContainer = document.getElementById('quizOptionsContainer');
            if (optionsContainer) {
                optionsContainer.innerHTML = `
                    <div class="segment-complete">
                        <div class="complete-icon">🎉</div>
                        <h3>${this.currentSegment.name} 完成！</h3>
                        <p>正确: ${this.correctCount} | 错误: ${this.wrongCount}</p>
                        <button class="training-action-btn" id="nextSegmentBtn">下一篇 →</button>
                    </div>
                `;
                optionsContainer.classList.remove('hidden');
            }
            
            const nextSegmentBtn = document.getElementById('nextSegmentBtn');
            if (nextSegmentBtn) {
                nextSegmentBtn.addEventListener('click', () => {
                    this.startSegment(this.currentSegmentIndex + 1);
                });
            }
        } else {
            this.showFinalResult();
        }
    }
}
```
